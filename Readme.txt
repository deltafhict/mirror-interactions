Builden van een vers gecloned project:
1. ga naar -> build -> configuration manager. Zet mirror voice op anycpu en sacknet op x64 en kruis build aan.
2. build nu het project
3. ga naar je references in het project, verwijder de sacknet reference. Doe nu rechtermuisknop op reference -> add reference -> browse naar de sacknet map -> bin -> x64 -> debug -> Sacknet.KinectFacialRecognition.dll toevoegen.
